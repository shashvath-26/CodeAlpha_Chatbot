# AI Chatbot

A rule-based AI Chatbot developed in Java as part of the CodeAlpha Java Programming Internship.

## Features
- FAQ-based question answering
- Keyword matching for responses
- Case-insensitive input handling
- Console-based interactive chat
- Built using HashMap and OOP

## Technologies Used
- Java
- HashMap
- Scanner
- String manipulation

## How to Run
1. Compile: `javac AIChatbot.java`
2. Run: `java AIChatbot`
